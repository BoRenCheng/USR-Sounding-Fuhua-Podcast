# 聲動福華：以 Podcast 銀齡共創課程促進社區跨世代交流

![Category](https://img.shields.io/badge/Category-USR_/_Social_Innovation-blue.svg)
![Field](https://img.shields.io/badge/Field-Intergenerational_Communication-orange.svg)
![Award](https://img.shields.io/badge/Award-Honorable_Mention-green.svg)

本專案為 **東吳大學學生社會實踐責任 (USR)** 之實踐成果，並榮獲 **佳作**。專案以「聲動福華」為核心，開發一系列專為銀髮族設計的 Podcast 共創課程，旨在透過新媒體科技打破數位鴻溝，促進大學生與社區長者間的跨世代對話 。

---

## 核心動機 
隨著高齡化社會來臨，長者的社會孤立與數位落差成為亟待解決的議題。本專案透過「銀髮主播體驗」方案，達成以下目標：
* **數位賦能**：引導長者學習錄音設備、Podcast 製作與數位敘事技巧。
* **跨世代交流**：建立「青銀共創」平台，由學生協助長者將生命經驗轉化為聲音紀錄。
* **社會參與**：透過聲音的傳遞，提升長者對社區的參與度與自我認同感。

---

## 專案執行重點 

### 1. 銀髮主播體驗課程
* 規劃適合高齡者的教學模組，涵蓋腳本發想、錄音軟體操作與訪談技巧。
* 成功帶領長者克服對科技的恐懼，產出具備溫度的聲音作品。

### 2. 跨世代共創機制
* 由學生擔任小隊輔，與長者進行一對一或小組互動，實踐青銀共學。
* 學生在過程中學習長者的生命智慧，長者則吸收年輕世代的創意。

### 3. 社區影響力
* 於福華社區實地執行，將長者的故事紀錄為數位資產，促進社區內的資訊流動與情感連結。
